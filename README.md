Cube3DViewer
============
This is a program that show the position and orientation of cube in 3D scene.  

## Features
* Receive pose from generator.
* Receive pose from file.
* Receive pose from socket.

## How to Use
```
$ ./cube3dviewer
```

How to use socket:  
start `cube3dviewer` first and start `test_client`.
